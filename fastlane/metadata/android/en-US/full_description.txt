Areada is a minimal offline Android reader focused on lightweight local reading.

It supports EPUB, TXT, and PDF files with a clean monochrome interface, local folder access, saved reading progress, bookmarks, and basic plain-text notes.

Features:
- EPUB, TXT, and PDF support
- Local plain-text notes
- Bookmarks and recent reading
- Saved reading progress
- Reader themes, font options, and font size settings
- EPUB chapter navigation
- PDF rendering through Android PdfRenderer
- Storage Access Framework folder picker
- Offline-only reading
- No internet permission
- No ads, analytics, tracking, accounts, or cloud sync
- No device-wide automatic scanning
